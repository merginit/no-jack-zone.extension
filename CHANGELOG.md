# Changelog

All notable changes to this project will be documented in this file.

## [0.0.4] - 2026-02-13

- Added `Ctrl+Tab` / `Ctrl+Shift+Tab` (next/previous tab).
- Added `Ctrl+Shift+T` (reopen closed tab), `Ctrl+Shift+N` (incognito window).
- Added zoom shortcuts: `Ctrl++`, `Ctrl+=`, `Ctrl+-`, `Ctrl+0`.
- Added `Ctrl+Shift+B` (bookmarks bar), `Ctrl+Shift+Delete` (clear data).
- Added `Ctrl+U` (view source), `Ctrl+E` / `Ctrl+K` (search bar).
- Added `Alt+D` (address bar), `Alt+Home` (home page), `Alt+F4` (close window).
- Added debatable shortcuts `Ctrl+B` (bookmarks) and `Ctrl+I` (page info) â€” disabled by default.
- Added `defaultEnabled` flag to shortcut definitions for finer default control.

## [0.0.3] - 2026-02-12

- Added `Ctrl+1` through `Ctrl+9` (switch to tab N / last tab) to protected shortcuts.

## [0.0.2] - 2026-02-11

- Rewrite of the store description in `STORE.txt` for better SEO.
- Added protection for advanced developer tools shortcuts:
  - Inspect (Ctrl+Shift+I)
  - Console (Ctrl+Shift+J)
  - Element (Ctrl+Shift+C)

## [0.0.1] - 2026-02-09

- Initial release (MVP)
